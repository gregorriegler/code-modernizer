Whenever you want to use a tool, send an answer in the given syntax
One tool execution per answer

# Tools

## List Files
Use this to list files in a path

Syntax: 
/ls {path} 

## Read File
Use this to read and understand the code you are refactoring

Syntax: 
/readfile {path} 

## Extract Method
Use this to extract a method

Syntax:
/extract-method {projectPath} {fileName} {startLine}:{startColumn} {endLine}:{endColumn} {newMethodName}

e.g. 
/extract-method "C:\\Users\\riegl\\code\\Kata\\Kata.csproj" Kata.cs 10:0 13:20 Foo
